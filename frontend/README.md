# 🖥️ Frontend

> 죽기 24시간 전에 — 채팅 인터페이스 (HTML + Vanilla JS)

---

## 📁 폴더 구조

```
Frontend/
├── index.html          # 메인 채팅 화면
├── style.css           # 전체 스타일 (루프 분위기 UI)
└── js/
    ├── main.js         # 진입점, 초기화
    ├── chat.js         # 채팅 전송 / 응답 렌더링
    ├── ui.js           # 심박수·노이즈 UI 트리거 (is_critical 처리)
    └── loop.js         # 루프 리셋 연출 (화면 노이즈 + 붉은 효과)
```

---

## 🎮 이 폴더에서 할 작업

### 1. 채팅 인터페이스 구현
- 캐릭터별 대화창 UI 구성 (김도현 · 차서연 · 엄마 · 이지우)
- 유저 입력창 + 전송 버튼
- 추천 메시지 버튼 3개 표시 (백엔드에서 GPT로 생성한 추천 발화 수신)

### 2. 백엔드 API 연동
- `fetch()`로 FastAPI 엔드포인트 호출
- 캐릭터 응답 텍스트 + 이미지 URL 수신 후 렌더링

### 3. 심박수 / 노이즈 UI 트리거
- 백엔드 응답에 `is_critical: true` 포함 시 화면 연출 발동
- 긴장감 연출: 화면 지직거림, 색상 변화 등 CSS 애니메이션 처리

### 4. 루프 리셋 연출
- 백엔드에서 루프 리셋 이벤트 수신 시 발동
- 화면에 붉은 피 튀김 효과 + 노이즈 → `Loop Reset` 문구 출력
- 루프 회차 카운터 UI 갱신

### 5. 최종 반전 연출 트리거
- 백엔드에서 반전 신호 수신 시 거울 장면 연출
- 고정 스크립트 재생 또는 GPT 생성 텍스트 출력 (백엔드와 협의)

---

## ▶️ 실행 방법

별도 빌드 없이 브라우저에서 바로 실행 가능합니다.

```bash
# 간단한 로컬 서버로 실행 (Python 설치 필요)
python -m http.server 3000
```

이후 브라우저에서 `http://localhost:3000` 접속

> ⚠️ 백엔드 서버(FastAPI)가 먼저 실행 중이어야 API 연동이 정상 작동합니다.

---

## 🔗 연동 백엔드 API 목록

| 메서드 | 엔드포인트 | 설명 |
|--------|-----------|------|
| `POST` | `/chat` | 유저 발화 전송 → 캐릭터 응답 수신 |
| `GET` | `/recommend` | 추천 메시지 3개 수신 |
| `GET` | `/image` | 대화 흐름에 맞는 이미지 URL 수신 |
| `POST` | `/loop/reset` | 루프 리셋 이벤트 수신 |

---

## ⚠️ 참고 사항

- 백엔드 API 주소는 `js/main.js` 상단의 `BASE_URL` 변수에서 관리
- 에러 발생 시 fallback 메시지를 화면에 표시하는 로직 필수 구현 (시연 대비)
- 이미지 자료는 프로듀서에게 수령 후 작업
